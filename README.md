# Atlanta-Food-Access-Dashboard

## Research Topic Description
This repository contains the source code for an interactive flexdashboard exploring spatial food access disparities in Fulton, DeKalb, and Clayton counties (Metro Atlanta). The analysis maps USDA-designated food deserts against 30-minute public transit accessibility to grocery stores and food pantries to identify critical neighborhood-level resource gaps. 

## Real-World Impact
This dashboard empowers local public health teams and nonprofit coordinators to rapidly identify severe "dual-deficit" food deserts that lack both grocery stores and alternative food pantries. By pinpointing these specific transit-isolated neighborhoods, policymakers can efficiently prioritize resource allocation and targeted outreach where interventions are most urgently needed.
